# 传统机器学习方法在手写数字图像分类中的应用

## 项目简介

本项目使用 sklearn 自带的 digits 手写数字数据集，实现了多种传统机器学习方法进行图像分类，并比较了不同模型的表现。通过本次实验，理解图像特征提取、模型训练与评估的基本流程。

## 数据集信息

| 项目 | 内容 |
|------|------|
| 数据集名称 | sklearn digits |
| 图像大小 | 8 × 8 像素 |
| 样本数量 | 1797 张 |
| 类别数量 | 10 类（数字 0-9） |
| 特征维度 | 64 维（8×8 展平） |

## 环境要求

```bash
python >= 3.8
numpy
matplotlib
scikit-learn
```

## 实验方法

本次实验使用以下 6 种传统机器学习方法：

| 方法 | 简要说明 |
|------|---------|
基于最近邻样本投票进行分类|
| 朴素贝叶斯 | 基于贝叶斯定理和特征独立性假设 |
| 逻辑回归 | 学习线性分类边界 |
| SVM | 寻找最大间隔分类超平面 |
| 决策树 | 根据特征阈值递归划分数据 |
| 随机森林 | 多棵决策树投票集成 |

## 实验结果

### 准确率对比

| 模型 | 测试准确率 |
|------|-----------|
| KNN | 0.9844 |
| 朴素贝叶斯 | 0.8289 |
| 逻辑回归 | 0.9622 |
| **SVM** | **0.9911** |
| 决策树 | 0.8244 |
| 随机森林 | 0.9600 |

**最佳模型**：SVM（准确率 99.11%）

### 混淆矩阵（SVM）

SVM 的混淆矩阵显示，最容易混淆的数字对是 **8 被误判为 1**（3 次），其次是 0→4、8→7、5→9 各 2 次。

### 各类别表现

| 数字 | 召回率 | 精确率 | 错误数 |
|------|--------|--------|--------|
| 8 | 0.8605 | 0.9487 | 6 |
| 1 | 0.9348 | 0.9348 | 3 |
| 9 | 0.9333 | 0.9333 | 3 |
| 0 | 0.9556 | 0.9773 | 2 |

数字 3、4、7 分类完全正确（错误数为 0）。


## 关键结论

1. **SVM 表现最佳**（99.11%），适合小样本高维图像分类任务；
2. **决策树和朴素贝叶斯表现较差**，前者容易过拟合，后者违反特征独立性假设；
3. **主要错误原因**：数字形态相似（如 8 与 1、5 与 9）、低分辨率导致细节丢失；
4. **原始像素特征的局限**：对平移、旋转、缩放敏感，缺乏几何不变性。

## 思考题摘要

- **为何要转换为特征向量？** 传统算法要求输入为固定长度数值向量，无法直接处理图像矩阵。
- **分类思想差异**：KNN（距离投票）、SVM（最大间隔）、决策树（阈值划分）、随机森林（集成投票）。
- **决策树过拟合原因**：不断分裂直到节点纯净，学习到噪声。
- **随机森林更稳定**：Bootstrap采样 + 随机特征 + 投票机制降低方差。
- **几何变换的问题**：平移/旋转/缩放会彻底改变像素分布，模型无法识别同一数字。
